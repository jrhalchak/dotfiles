# Org Vault

## Mac Emacs
Using emacs-mac port, with the following build options:

```bash
brew reinstall railwaycat/emacsmacport/emacs-mac \
  --with-modules \
  --with-native-compilation \
  --with-no-title-bars \
  --with-imagemagick \
  --with-librsvg \
  --with-xwidgets \
  --with-unlimited-select \
  --with-mac-metal
```
